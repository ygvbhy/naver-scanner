# 네이버 쇼핑 최저가 검색 웹 프로그램

네이버 쇼핑 기준 오픈마켓 최저가를 확인하는 도구입니다.

## 기술 스택

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 변수 설정

`ENV_SETUP.md` 파일을 참고하여 네이버 API 인증 정보를 설정하세요.

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 주요 기능

- 네이버 쇼핑 상품 검색
- 최저가 기준 정렬
- 검색 결과 리스트 표시
